delivery
 
drivers

Customers

The

The  system  relies  on  a  relational  database  to  manage  structured  data  such  as  customers,  orders,  
menu
 
items,

deliveries.
 
It
 
ensures
 
data

2.  ENTITIES  WITH  ATTRIBUTES  (WITH  DATA  TYPES)  
  
 Customer  
Attribute  Data  Type  Description  
customer_id  INTEGER  (PK)  Unique  customer  ID  
name  VARCHAR(20)  Customer  name  (Text  ≤  20  chars)  
phone  VARCHAR(15)  Contact  number  
address  VARCHAR(255)  Address  (Text  ≤  255  chars)  
created_at  DATE  Account  creation  date  
status  CHAR(1)  ('A','I')  (Single  char  limited)  
  
 
 
Restaurant  
Attribute  Data  Type  Description  
restaurant_id  INTEGER  (PK)  Unique  restaurant  ID  
name  VARCHAR(20)  Restaurant  name  (Text  ≤  20  chars)  
cuisine_type  VARCHAR(20)  ('Italian','Asian','FastFood')  (Limited  text  choices)  
opening_time  DATE  Opening  time  
closing_time  DATE  Closing  time  
status  CHAR(1)  ('A','I')  (Single  char  limited)  
  
Menu_Item  
Attribute  Data  Type  Description  
item_id  INTEGER  (PK)  Unique  item  ID  
restaurant_id  INTEGER  (FK)  References  Restaurant  (Foreign  Key)  
name  VARCHAR(20)  Item  name  (Text  ≤  20  chars)  
price  DECIMAL(8,2)  Item  price  (Decimal)  
category  VARCHAR(20)  ('Meal','Drink','Dessert')  (Limited  text)  
is_available  CHAR(1)  ('Y','N')  (Single  char  limited)  
 Order  
Attribute  Data  Type  Description  
order_id  INTEGER  (PK)  Unique  order  ID  
customer_id  INTEGER  (FK)  References  Customer  
restaurant_id  INTEGER  (FK)  References  Restaurant  
order_time  DATE  Order  timestamp  (Date)  
status  VARCHAR(20)  ('Pending','Preparing','Delivered','Cancelled')  (Limited  text)  
total_amount  DECIMAL(10,2)  Total  order  cost  (Decimal)  
  
 Order_Item  
Attribute  Data  Type  Description  
order_id  INTEGER  (FK,  PK)  
References  Orders  
item_id  INTEGER  (FK,  PK)  
References  Menu_Item  
quantity  INTEGER  Quantity  ordered  
unit_price  DECIMAL(8,2)  Price  per  unit  
subtotal  DECIMAL(10,2)  Calculated  subtotal  
  
 
 Driver  
Attribute  Data  Type  Description  
driver_id  INTEGER  (PK)  Unique  driver  ID  
name  VARCHAR(20)  Driver  name  (Text  ≤  20  chars)  
status  CHAR(1)  ('A','B')  (Single  char  limited)  
total_deliveries  INTEGER  Total  deliveries  (Integer)

 Delivery

Payment
